## Day 2

学习内容：
- if / elif / else 条件判断
- 比较运算（>=, <=, ==）
- input 输入处理
- 基本异常处理（try / except）

完成内容：
- 实现了基于年龄的身份判断（儿童 / 青少年 / 成年人）
- 实现了基于学习目标的分支判断（AI / 编程 / 其他）
- 将程序整合为完整的输入 → 判断 → 输出流程
- 使用 f-string 优化输出格式

问题与解决：
1. 年龄输入异常后程序仍继续执行  
   - 原因：try/except 只捕获错误，没有阻止流程继续  
   - 解决：使用 while True + try/except，实现错误后重新输入  

2. goal 无法识别大小写（AI vs ai）  
   - 原因：字符串比较区分大小写  
   - 解决：使用 .lower() 统一转为小写  

3. 输入可能包含空格  
   - 解决：使用 .strip() 去除首尾空格  

关键收获：
- 程序不能假设用户输入一定正确（需要做输入校验）
- 条件判断是程序逻辑的核心结构
- 可以先将判断结果存入变量，再统一输出（代码更清晰）
- 字符串处理（strip / lower）在实际开发中很常见

当前掌握程度：

- 能处理简单用户输入异常
- 能构建简单的交互式程序

待加强点：

- 条件逻辑复杂时需要进一步练习
